# 🚀 SkillSwapeX

SkillSwapeX is a full-stack web platform that enables users to **exchange skills, request learning sessions, and collaborate on freelance work**.  
It integrates a **credit-based system** where users earn and spend virtual credits based on session participation.

---

## 🌟 Key Features

### 🔐 Authentication & Authorization
- Role-Based Authentication (User / Admin)
- Secure Login & Registration
- Token-based authentication

### 👥 User Functionalities
- Create and manage user profiles
- Send requests to:
  - 📚 Learn a skill
  - 💼 Request freelance work
- Accept / Reject incoming requests

### 🎯 Skill Exchange System
- Users can offer skills to others
- Choose **learning sessions (1 – 100)**
- Support for:
  - 🌐 Online sessions
  - 📍 Offline sessions

### 💰 Credit System
- Virtual credit transfer mechanism
- Earn credits based on sessions attended
- Spend credits to learn new skills

### 📦 Additional Features
- Request tracking system
- Dashboard for users
- Clean and responsive UI
